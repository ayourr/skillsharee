# Skill Sharee API
NodeJS API for skill sharee App.

### Entities
1 - User
2 - post
3 - post's comment
4 - tag
5 - page
6 - events

### Getting Started
I use babel Node for base of project. and [this tutorials](http://scottksmith.com/blog/2014/05/02/building-restful-apis-with-node) for create API.
